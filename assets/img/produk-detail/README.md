# Foto Halaman Detail Produk

Folder ini untuk **foto masalah & solusi** di halaman `produk-detail.html?slug=...`.

- Aspect ratio: **16:9** (suggested 1200 × 675 px)
- Format: JPG quality 80-90% (compress di TinyPNG/Squoosh, target <200 KB)
- Filename **case-sensitive** — copy persis dari list di bawah

Kalau file sudah ada di folder ini, otomatis nimpa placeholder di website. Kalau belum ada, dashed placeholder dengan nama file akan tetap muncul (gak perlu edit HTML).
